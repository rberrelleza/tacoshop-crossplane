# Create a Development Environment with okteto, Kubernetes, and Cloud Services

This is an example of how to configure and deploy a development environment that includes polyglot microservices, an AWS SQS queue, and an S3 bucket. This sample uses Okteto and Crossplane to show you how to build a modern development workflow in Kubernetes
